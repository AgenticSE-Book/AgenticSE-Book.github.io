# Agentic Software Engineering - The Future of Code

This repository contains the website for the book **Agentic Software Engineering** by Ahmed E. Hassan.

## Overview

Discover how autonomous AI agents are revolutionizing the way we build, test, and deploy software. This is the definitive guide to the next era of engineering.

The website provides multiple ways to consume the book's content:
- 📖 **Read Online**: View the PDF directly in your browser.
- 📥 **Download PDF**: Download the PDF version for offline reading.
- 🎧 **Listen on the Go**: Experience a 27-minute AI-generated audio summary that explores the core concepts, the shift from coding to architecture, and the future of software development.
- 🌐 **Multilingual**: The website natively supports both English and Chinese.

## What You'll Learn

* **The Real Bottleneck**: Code production isn't the bottleneck. The real challenge is complexity, communication, and maintaining system integrity over time.
* **The Discipline**: Agentic Software Engineering is the discipline of ensuring reliability and trust from stochastic AI and human contributors.
* **The New Winners**: Success belongs to teams who set clear intent, manage risk boundaries, and demand evidence—not just those who type fastest.
